Transformer-Based Stock Price Forecasting

Stock Price Forecasting is my final project for my Deep Learning class at the University of Illinois at Springfield. It is currently a work in progress, the project deadline is May 1st. For this project, I will be tackling the problem of predicting stock price using the "Huge Stock Market Dataset" available on Kaggle. The key variables include Date, Ticker, Open, High, Low, Close Volume. The end goal is to predict the future stock price, specifically the Close price for the next trading day, using historical values as predictors. I plan to use a Transformer-based model architecture for this time series forecasting problem.

Solving the Almost Shortest Path Problem with Dijkstra's Algorithm

This "project" focused on implementing Dijkstra's Algorithm in Java to solve the Almost Shortest Path problem. The implementation included creating a Heap-Based Priority Queue and applying the algorithm to a graph representation. Key steps involved finding the shortest path, removing critical edges, and then recalculating to find the almost shortest path. The project highlighted the complexities of algorithm implementation and the importance of data structure proficiency.

Academic Performance Tracking Database

This "project", undertaken by a team of three, focused on designing a Course Performance Information Management System. The system is intended to record and manage student performance data in various courses, enabling the input and storage of marks, the dynamic addition of assessments, and the calculation of weighted grades. The goal was to create a streamlined and flexible solution to improve the efficiency and accuracy of student performance management in educational settings. The team collaborated to address the need for a modern system, moving away from traditional cumbersome methods.
